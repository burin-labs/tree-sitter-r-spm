# tree-sitter-r-spm

Swift Package Manager wrapper for the [tree-sitter-r](https://github.com/r-lib/tree-sitter-r) grammar.

## Usage

Add this package to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/burin-labs/tree-sitter-r-spm.git", from: "0.1.0")
]
```

Then add `TreeSitterR` as a dependency of your target.
